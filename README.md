# R Data Analysis and Visualization Collection

This repository contains a collection of R scripts, datasets, and generated outputs used for data analysis, statistical practice, and visualization. The project demonstrates fundamental to intermediate R programming concepts including data manipulation, exploratory analysis, and plotting.

---

## Overview

The purpose of this repository is to:

* Practice data analysis using R
* Explore statistical concepts and datasets
* Generate visualizations and reports
* Serve as a reference for R-based workflows

## Contents

### R Scripts

* **Pavia.R** – Main analysis script (likely coursework or project-based)
* **demo.R** – Demonstration of basic R functionality
* **lungcap.R** – Analysis of lung capacity dataset
* **prac.R** – Practice scripts for various operations
* **review.R** – Review or summary analysis
* **tempCodeRunnerFile.R** – Temporary file generated by IDE (can be ignored)

### Datasets

### Outputs

## Requirements

* R (version 4.0 or higher recommended)
* RStudio or any R-compatible IDE (optional but recommended)

---

## How to Run

1. Open R or RStudio
2. Set the working directory to the project folder:

```r
setwd("path/to/project")
```

3. Run any script:

```r
source("Pavia.R")
```
